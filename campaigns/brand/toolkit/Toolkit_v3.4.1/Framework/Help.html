Help

Where it may assist users to complete a form, one of three different levels of help information can be applied.

  • Help levels 1 and 2 relate specifically to individual or groups of related form fields.
  • Help level 3 is to be used for help information that is useful but ancillary to the form or tool.

Level 1 - Help note displayed on the page

  • Provides a short note to help the user understand how to complete the field.
  • Often in the form of an example eg 'PX 1234 56 7890'.
  • Is written in brief and succinct way.
  • Should not be overused on a form. Overuse can make the page look cluttered.
  • Placed beneath the input or set of inputs it relates to.

Level 1 help

Level 2 - Help revealed via javascript

  • Use level 2 help if more information is required above what Level 1 help one can offer.
  • Is specific to one input field or a group of related input fields.
  • Is accessed via the 'Help' link.
  • The help link must be associated via visual proximity, near the field to which it refers.
  • Upon clicking 'Help', a white box appears inside the grey keyline box, underneath the question it pertains to.
  • The help box appears with a title within the white box.
  • Once revealed, the original 'Show help' link dynamically changes to 'Hide help'.
  • Text should concisely aid the user in completing the related field(s).
  • If javascript is disabled, the help link is not displayed and the Help box is displayed as default.

Help revealed

Level 3 - Separate help page for righthand side help links

  • Used for ancillary supporting help that is not directly related to a specific form field. Ancillary help is positioned in the fixed width right hand column under the heading 'Help with this page'.
  • Upon selecting a level 3 help link, a new help page is opened in a new window.
