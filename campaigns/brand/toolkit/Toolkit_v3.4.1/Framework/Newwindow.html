Opens new window

The toolkit supports various sizes of new window openings for links. Popup window techniques are handled with Javascript to adhere to the rules outlined with a strict doctype.

To automatically create an opens new window link add a class of "newwindow" to the anchor.

The script plugin file "dg.popup.js" must also be included in the head of the document.

Popup windows may open up at 800 x 600, 640 x 480 and at default size. The 800 x 600 and 640 x 480 windows will open with toolbars by default, however you may also open these popup windows with no toolbars.

To control the parameters of the new windows add one the following "rel" tags to the anchor:

  • window800
  • window640
  • window800_notools
  • window640_notools
