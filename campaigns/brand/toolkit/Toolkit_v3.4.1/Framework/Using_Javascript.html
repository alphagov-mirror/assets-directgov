Using Javascript

Javascript can be used to deliver functionality throughout the tools and transactions but should not be relied on alone.

You should always include an alternative method of delivering the functionality and content. If advanced script is used then a basic level should also be included, delivered using conditional logic. To accommodate browsers which may have Javascript disabled you must include content using the <noscript> tag.

Do not use Javascript to process any type of personal data.

Ensure that your Javascript is contained within an external javascript (.js) file and include at runtime.

When using Javascript within links ensure that you include an alternative to access the URL as well as using an event handler to invoke the Javascript code.
