Link styles

The link styles below refer to links within the body of the page or within the right hand column. Within the main content of the page a link will be on a white background.

Links style within the whitespace of the page

Upstate

All body copy inline links which sit on a white background within the page will be default Directgov black #000000 with an underline. Bullet and inline links are permitted. Bullet links will be the Directgov orange #FF5B00 and should include the orange (#FF5B00) left hand arrow chevron with no underline.

Headings should not normally be used as a link.

Colour

For body copy use #000000.

For bullets use #FF5B00.

Hover state

The hover state of an inline link is #FF5B00, underline.

The hover state of a bullet link is underline.

Visited links

Visited inline links should use undeline, #666.

Visited toggle links should use #FF5B00.

Visited bullet links should use #BF4400.

Links style within the right hand column

Upstate

All links within the right hand column will only underline on hover but will contain the left hand arrow chevron.

Colour

For body copy use #000000.

Hover state

The hover state of a link is underline.

Visited links

Visited links should use #666.

Lists of links

Any lists of links will have an arrow chevron to the left hand side of the text.

Error links

Upstate

All linking to error messages will have a red, #F00000 chevron arrow to the left hand side and no underline.

Colour

For body copy use #F00000.

Hover state

The hover state of a link is underline and red in colour, #F00000.

Visited links

Visited links should use no underline and red in colour, #F00000.

Scale

Font sizes = ems, to the scale: 1em = 10px/pt
