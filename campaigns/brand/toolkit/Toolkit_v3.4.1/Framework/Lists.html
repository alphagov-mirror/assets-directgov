Lists

Lists can be used to display an ordered group of results - search results being an obvious example.

Displaying results in a list

Requirements

  • A list title should be clearly displayed.
  • Each list column should be clearly labelled.
  • Lists may have selectable or non-selectable information. It must be obvious which items are selectable and which aren't.
  • Any list items which are selectable should be displayed in orange.
  • Numbers of results returned can be displayed at the top of the page.
  • The franchise may deem it appropriate to display column headings within the list. Column headers could be sortable via either a dropdown selection box and go button at the top of the list of results, or via selectable column headings.
  • It may be deemed necessary to reduce the number of results displayed on one page. This is at the discretion of the franchise to determine if paging is required or desired.
  • A key can be provided if icons have been used.
  • On clicking selectable information a detail page may be presented.
