Directgov toolkit v3.5

Getting started with Directgov toolkit v3.5

This site provides guidance and suggestions to enable off-platform services to conform to the Directgov toolkit, so that citizens understand and become increasingly comfortable transacting with Government online.

Please familiarise yourself with the toolkit in order to apply the Directgov brand to your application.

This guidance is to be used alongside discussions with the Directgov Publishing team and is continually evolving, so please check regularly for updates. If you have any questions or points you wish to clarify please don't hesitate to get in touch.

What is the toolkit?

The Directgov toolkit is a resource for designers and developers involved in the delivery of services through the Directgov brand and not through the Directgov infrastructure.

What are tools and transactions?

Tools and transactions are standalone applications, hosted and run outside of the Directgov core infrastructure.

Before you start using the toolkit

Please ensure that you copy any code from the live examples provided and not these framework pages.
