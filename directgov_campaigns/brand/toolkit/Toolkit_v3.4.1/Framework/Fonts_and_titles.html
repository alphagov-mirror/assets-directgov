Fonts and titles

The Directgov font system has been designed to allow flexibility in typographic layout.

Font family

  • The default Directgov online font is Verdana. So the font family within the style sheet is set to Verdana, Arial, Helvetica, sans-serif.

Colour

  • For body copy use #000000.
  • For section headings use #333333 or #FF5B00 for H1 tags.

Line height

Always set the line height at 1.5em greater than the type size. For example 1.2em font/1.5em leading.

Alignment

Always left-align copy.

Tracking

Do not use tracking with Verdana online.

Scale

  • The base font size is set to 62.5% in the stylesheets only
  • In technical terms: Declaring 62.5% is the most consistent way in acheiving cross-browser results equal to 1em or 10px
  • The on-screen paragraph base font size is 12px (or 1.2em in the stylesheets)
  • All headings and text items are re-sized relatively using em units
