Three column layout

The layout grid is semi-fluid. The maximum width is set at 1210 pixels and reduces down to a minimum of 760 pixels.

In the three column layout the outer columns have a static width and the centre column is fluid.

General principles of zone use

General principles of the three column layout

  • Utilises a left hand column, a right hand column and a central section column.
  • Left and right hand columns are fixed width while the middle column is fluid.
  • Left hand or right hand column can be left blank if needs to be to provide alignment.
  • Left hand column to be used for step tracker/process indication or narrow-by/filtering functionality only.
  • Right-hand column to be used for co-branding, ancillary help etc.
  • Central section to be used for the tool or service.
